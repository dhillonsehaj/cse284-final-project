Results Guide — PLINK2 IBD Benchmarking Project
================================================

All results are in the results/ directory of the project.
Input data: 230 samples, 118,164 QC'd biallelic variants on chromosome 22.


1. plink_king.kin0
------------------
Source:    PLINK2 v2.0.0-a.7 M1 --make-king-table
Format:   Tab-delimited text, one row per sample pair
Rows:     26,335 (all choose-2 pairs of 230 samples)
Columns:
  #FID1    - Family ID of sample 1 (all "0" in this dataset)
  IID1     - Individual ID of sample 1
  FID2     - Family ID of sample 2
  IID2     - Individual ID of sample 2
  NSNP     - Number of variants used for the pair (118,164 for all)
  HETHET   - Proportion of variants where both samples are heterozygous
  IBS0     - Proportion of variants where samples are opposite homozygotes
  KINSHIP  - KING-robust kinship coefficient

Use for:   Identifying related pairs and classifying degree of relatedness.
Thresholds:
  > 0.354        → duplicate / MZ twin
  0.177 – 0.354  → 1st degree (parent-child, full siblings)
  0.0884 – 0.177 → 2nd degree (half-siblings, grandparent, avuncular)
  0.0442 – 0.0884→ 3rd degree (first cousins)
  < 0.0442       → unrelated


2. chr22_genome.genome
----------------------
Source:    PLINK 1.9.0-b.7.11 --genome
Format:   Space-delimited text (fixed-width), one row per sample pair
Rows:     26,335
Columns:
  FID1     - Family ID of sample 1
  IID1     - Individual ID of sample 1
  FID2     - Family ID of sample 2
  IID2     - Individual ID of sample 2
  RT       - Relationship type (OT = other/unrelated, based on FID)
  EZ       - Expected IBD sharing given RT (0 for OT)
  Z0       - Estimated proportion of genome with IBD = 0 (no alleles shared)
  Z1       - Estimated proportion of genome with IBD = 1 (one allele shared)
  Z2       - Estimated proportion of genome with IBD = 2 (both alleles shared)
  PI_HAT   - Overall proportion of IBD sharing = Z1/2 + Z2
  PHE      - Phenotype (-1 = missing)
  DST      - IBS distance (genome-wide similarity, range 0–1)
  PPC      - IBS p-value testing whether the pair is more related than expected
  RATIO    - HETHET / IBS0 ratio

Use for:   Distinguishing relationship types via IBD proportions.
           Comparable to GERMLINE's per-pair IBD summary metrics.
Expected patterns:
  Parent-child:   Z0 ≈ 0,    Z1 ≈ 1,    Z2 ≈ 0,    PI_HAT ≈ 0.5
  Full siblings:  Z0 ≈ 0.25, Z1 ≈ 0.5,  Z2 ≈ 0.25, PI_HAT ≈ 0.5
  Half-siblings:  Z0 ≈ 0.5,  Z1 ≈ 0.5,  Z2 ≈ 0,    PI_HAT ≈ 0.25
  First cousins:  Z0 ≈ 0.75, Z1 ≈ 0.25, Z2 ≈ 0,    PI_HAT ≈ 0.125
  Unrelated:      Z0 ≈ 1,    Z1 ≈ 0,    Z2 ≈ 0,    PI_HAT ≈ 0


3. Log files
------------
  plink_king.log           - PLINK2 log for KING kinship run
  chr22_genome.log         - PLINK 1.9 log for --genome run
  chr22_genome.nosex       - List of samples with ambiguous sex (all 230)
  plink_ibd.log            - Earlier IBD attempt log
  chr22_ibd_segments.log   - Failed --segment attempt log (flag retired in PLINK 1.9)


Key difference vs. GERMLINE
----------------------------
PLINK --genome estimates Z0/Z1/Z2/PI_HAT via method-of-moments from genome-wide
allele sharing counts. GERMLINE detects actual contiguous shared haplotype segments
and sums their lengths. Both produce per-pair IBD proportions, but GERMLINE also
reports individual segment positions and lengths. PLINK's method is faster but has
less power for distant relatives (beyond 3rd degree).
